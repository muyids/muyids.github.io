# 夏训初步计划(6-7-8月)

# 一、训练时间：早5: 30-7: 30、晚17: 30-22: 00

# 二、训练目标：十公里43分、半马145

# 三、训练方法：

# 1、间歇跑
  • 400-800米 配速330-340
  • 1000-2000米 配速345-355
  • 3000米 配速420-435
  • 5000米 配速430-450
  • ST 配速320-340
# 2、节奏跑

5k-10k-15k-21k 配速 420-550

# 3、配速跑

5k-21k 配速430-540

# 4、LSD、有氧跑、心率跑、混氧跑等
# 5、力量训练
# 6、同时结合丹尼尔斯经典跑步训练法-半马训练课表

# 四、训练时长：30-150分

# 五、训练环境：间歇跑去南长河公园塑胶跑道、其它路跑或尽量选择带胶道，周六去奥森。

# 六、训练频次：4-5/周

# 七、训练负荷：50k左右/周

# 八、具体课表见日历

# 避开高温，及时补水，预防受伤。
